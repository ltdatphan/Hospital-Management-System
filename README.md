# Hospital Management System

### Requirements
Node = 18.4.0 

npm = 8.12.1
### Installation

```zsh
# Clone the repo
$ git clone https://github.com/ncheungg/Hospital-Management-System.git

# Go into the root folder
$ cd Hospital-Management-System

# Install packages
$ npm install
```

### Start local server

```zsh
$ npm run dev
```
